# Regex Tutorial

## Description

This tutorial allows one to understand RegEx, short for Regular Expressions.  No challenges were faced in creating this tutorial.  In the future, I would like to make this tutorial more easy to understand.

## Installation

To view this tutorial, one must do a git clone from the repository. 

## Usage

One can view this tutorial to gain a better understanding of RegEx.
